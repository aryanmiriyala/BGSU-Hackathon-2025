# Travel Advisory App

## Prerequisites

Install the following:

- Node.js (v16+)
- npm or Yarn

## Installation Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/my-react-app.git
cd my-react-app
```

2. Install dependencies:

```bash
npm install
# or
yarn
```

3. Install map libraries:

```bash
npm install react-simple-maps d3 topojson
# or
yarn add react-simple-maps d3 topojson
```

4. Run the development server:

```bash
npm run dev
# or
yarn dev
```

App will run at: http://localhost:5173
